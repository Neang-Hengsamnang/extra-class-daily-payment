📤 នាំទិន្នន័យសិស្សចេញជា Excel

ទាញយកទិន្នន័យសិស្សទាំងអស់ (ID, Full Name, Gender, Grade Level) ជា file Excel មួយ។ ប្រើវាជា File Backup មួយ ឬសម្រាប់រៀបចំទិន្នន័យក្នុងការនាំចូល។

ទាញយកទិន្នន័យសិស្សជា Excel
📥 នាំទិន្នន័យសិស្សចូល

ការណែនាំ:

  • សូម upload ជា file excel មួយ (.xlsx or .xls) ជាមួយជួរឈរ (headers are case‑insensitive): id, full_name, gender (optional), grade_level (optional).
  • យ៉ាងហោចណាស់ id និង full_name គឺទាមទារត្រូវតែមាន ក្នុងជួរដេកនីមួយៗ។
  • Student IDs: ប្រសិនបើ ID មានរួចហើយ នោះទិន្នន័យនឹង ត្រូវបានធ្វើបច្ចុប្បន្នភាព។ ផ្ទុយមកវិញ, ទិន្នន័យថ្មីនឹងត្រូវបានបង្កើត។
  • Gender គួរតែជា "ប្រុស" or "ស្រី" (ឬទុកទទេរក៏បាន)។
  • Grade Level អាចជាអក្សរ (ឧ. "ថ្នាក់ទី 10")។
  • Default Course (optional) ត្រូវតែមានឈ្មោះដូចគ្នាជាក់ស្ដែងទៅនិងទិន្នន័យវគ្គសិក្សាដែលមានស្រាប់។ ប្រសិនបើរកមិនឃើញ វានឹងបន្សល់ទុកនៅឱ្យទទេ។
